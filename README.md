Página Web Portafolio de Mariana Hernández

Descripción: Este proyecto crea una página web para el portafolio profesional de Mariana Hernández. El proyecto tiene como objetivo crear un sitio web con su información profesional, incluyendo una breve descripción, CV, trayectoria, proyectos y links importantes. Que pueda servir como referencia para contactos interesados en trabajar con Mariana.

Fucionalidad: La creación de la página web se realizó utilizando el lenguaje html de marcado para la estructura de la página. Para el diseño de la página se utilizó el lenguaje de estilos CSS. Mediante técnicas avanzadas de media queries se desarrolló un diseño responsivo capaz de adaptar a diferentes pantallas.

Mejoras del proyecto: Este proyecto aun cuenta con limitantes,

Los links no tienen una url válida.
El proyecto personal no tiene la información actualizada.
La información proporcionada es de un personaje imaginario - idealmente sería destinado a una persona real y en especifico a mí como desarrolladora de la página.

Para ofrecer un correcto funcionamiento de la página web se necesitaría al menos desarrollar los puntos sugeridos anteriormente.
